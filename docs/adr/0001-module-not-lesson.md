# Rename teach's "lesson" to "module"

`/teach` calls its teaching unit a "lesson," numbered and stored in `./lessons/`. We're renaming it to "module" (`./modules/`) throughout learn-and-tell.

"Lesson" implies a linear curriculum taught by an authority — the opposite of learn-and-tell's positioning (non-linear, problem-posing driven, user controls the path). Keeping "lesson" would make every doc and template quietly fight the product's own pitch. The cost of reversing this later is real (touches file naming, SKILL.md, templates) but paid once, up front, rather than accumulating as inconsistent terminology across a growing skill.
