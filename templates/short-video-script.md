# Short Video Script Template

For 30s–3min content: one sharp highlight, minimal setup. See `references/OUTPUT-TEMPLATES-GUIDE.md` for when this format actually fits.

```md
# {Working title}

**Source highlights**: {link the ./highlights/*.md files this draws on}
**Target length**: {30s / 60s / 90s}

## Hook (first 3 seconds)
{The single most surprising claim or question — state it cold, no preamble.
This is the reframe from the highlight itself, sharpened.}

## Payoff (the middle)
{One causal chain or reveal that resolves the hook. Not a list of facts —
the *why*. Keep this to what fits in the target length; cut anything
that isn't load-bearing for the hook's resolution.}

## Button (last line)
{A line that closes the thought — often the highlight's own words,
lightly polished. Optionally a question back to the viewer.}

## Visual notes
{Which ./images/ assets fit as-is vs need regenerating at publish quality.
Note pacing: where a cut/beat should land relative to the script.}
```

## Rules

- One highlight, one hook. Don't try to fit two ideas — that's what the blog/long-video formats are for.
- No citation clutter in the script itself — sourcing lives in the module/RESOURCES.md, not the final script.
- Match the user's observed narrative preference (`learning-style.md`) — causal-chain framing over flat fact statements, unless observation says otherwise.
